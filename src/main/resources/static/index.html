Concurrency Patterns POC

Java 21 + Spring Boot + virtual threads. Spam the buttons and watch what each pattern does.

1. Rate Limiting

Token bucket (capacity 5, refill 2/sec): allows bursts of 5, then sustains 2/sec.
Sliding window (5 req / 1 sec): strict — never more than 5 in any 1-second window.

2. Debounce & Throttle

Debounce (400ms quiet): fires once, 400ms after you stop typing.
Throttle (1000ms): fires at most once per second; extra clicks are dropped.
